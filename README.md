## 🚀 Tecnologias & Metodologias

Este projeto utiliza uma infraestrutura moderna de Ciência de Dados:

* **Linguagens:** Python (Polars/Scikit-Learn) e R (Tidyverse).
* **Performance:** Utilização de **Polars** para processamento massivo de dados (POS Data).
* **Publicação:** Quarto 1.8 para geração de relatórios técnicos e blogs.
* **Infraestrutura:** Ambiente isolado em **Docker** rodando no Linux (CachyOS/BigLinux).
* **Segurança:** Versionamento via Git com autenticação SSH (ED25519).
